Expand description

Things we can encode and read from a Reader, Sized version. Easier to work with when types are instantiated, for example in protocol crates such as tlspuffin.

Required Methods§

source

fn encode(&self, bytes: &mut Vec<u8>)

Encode yourself by appending onto bytes.

source

fn read(_: &mut Reader<'_>) -> Option<Self>

Decode yourself by fiddling with the Reader. Return Some if it worked, None if not.
