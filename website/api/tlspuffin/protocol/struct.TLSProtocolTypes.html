source§

fn extract_knowledge<'a>( &'a self, knowledges: &mut Vec<Knowledge<'a, TLSProtocolTypes>>, _: Option<TlsQueryMatcher>, source: &'a Source, ) -> Result<(), Error>

Extracts knowledge from a crate::tls::rustls::msgs::message::Message. Only plaintext messages yield more knowledge than their binary payload. If a message is an ApplicationData (TLS 1.3) or an encrypted Heartbeet or Handhake message (TLS 1.2), then only the message itself and the binary payload is returned.
